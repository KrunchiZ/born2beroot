A project learning how to setup a VM using Oracle VirtualBox.

### Tools I Used
| Commands | Used to Get                   |
| -------- | ----------------------------- |
| uname    | OS Architecture & Kernel info |
| lscpu    | Physical CPU count            |
| nproc    | Virtual CPU count             |
| free     | RAM usage                     |
